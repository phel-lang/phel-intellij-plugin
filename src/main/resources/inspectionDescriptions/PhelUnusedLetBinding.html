Reports a binding in a let-like form whose name is never used.

A binding name is flagged when it appears in neither the body of the form nor the value of any later binding. Phel binds sequentially, so a value can only refer to names introduced before it; both places are taken into account.

Applies to the binding-introducing forms let, if-let, when-let, loop, for, foreach, binding and dofor.

Example:


(let [x 1
      y 2]   ;; 'y' is reported: Binding 'y' is never used.
  (println x))

Ignored names

Names conventionally used to discard a value are never reported: _, any name starting with _, and names starting with & (rest markers).

A quick-fix removes the unused binding pair from the binding vector.
