Reports usage of deprecated Phel functions.

When a Phel function has been deprecated, this inspection highlights its usage and suggests replacing it with the recommended alternative.

Features:

  • Detects deprecated functions from the Phel standard library
  • Shows deprecation version and replacement suggestion in the warning message
  • Provides a quick-fix to automatically replace deprecated functions
  • Automatically adds required namespace imports when the replacement is from a different namespace

Example:


;; Before (deprecated)
(str-contains? "hello" "ell")

;; After (using quick-fix)
(str/contains? "hello" "ell")

The quick-fix will also add (:require phel\str) to your namespace declaration if needed.
